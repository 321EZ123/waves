## Deploying Waves on your own server:
> [!NOTE] 
> A subdomain or a domain is **required** for this.

1. Login your server
2. Go to your server terminal
3. Run the commands bellow
   
```bash
sudo git clone https://github.com/xojw/waves

cd waves

sh setup.sh
```
